# 网文常见问题库

## 一、情节逻辑问题

### 1.1 常见矛盾类型

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 时间矛盾 | 时间线前后不一致 | 第3章说"三天后"，第4章却出现"第二天" |
| 地点矛盾 | 人物位置不合理 | 上一章还在A城，下一章突然到了B城 |
| 能力矛盾 | 人物能力突然变化 | 前10章不会魔法，第11章突然使用魔法 |
| 物品矛盾 | 物品凭空出现或消失 | 之前从未提及的道具突然派上用场 |
| 关系矛盾 | 人物关系前后不一致 | 上一章是仇人，下一章变成朋友 |

### 1.2 常见逻辑漏洞

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 无因之果 | 事件没有原因就发生 | 主角突然获得力量，没有任何铺垫 |
| 无果之因 | 事件发生后没有结果 | 埋下的伏笔再也没有提及 |
| 动机不足 | 人物行为缺乏动机 | 反派为了破坏而破坏，没有深层原因 |
| 巧合过度 | 关键事件靠巧合推动 | 主角每次遇险都刚好有人相救 |
| 规则突破 | 突破世界观规则却不解释 | 力量体系设定有明确限制，却被随意突破 |

### 1.3 常见因果链问题

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 因果断裂 | 事件之间没有因果关系 | 前一段在吃饭，后一段突然在战斗 |
| 因果倒置 | 原因在结果之后 | 因为受伤所以打架 |
| 因果薄弱 | 因果关系不明显 | 主角因为心情不好所以变强 |

## 二、人物一致性问题

### 2.1 常见OOC行为

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 性格突变 | 人物性格突然改变 | 一直冷静的主角突然冲动行事 |
| 智商波动 | 人物智商忽高忽低 | 聪明的主角做出愚蠢的决定 |
| 能力波动 | 人物能力不稳定 | 强时无敌，弱时不堪一击 |
| 态度突变 | 人物态度突然转变 | 讨厌某人突然变成喜欢 |
| 行为反常 | 人物做出不符合身份的行为 | 贵族做出粗俗举动 |

### 2.2 常见成长问题

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 成长过快 | 人物成长速度不合理 | 一章之内从新手变成高手 |
| 成长过慢 | 人物成长停滞不前 | 几十章过去没有任何变化 |
| 成长无因 | 人物成长没有原因 | 没有经历任何事件就变强了 |
| 成长不符 | 成长方向不符合性格 | 胆小的人突然变得勇敢 |

### 2.3 常见关系问题

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 关系突然 | 关系变化过于突然 | 陌生人突然变成亲密朋友 |
| 关系矛盾 | 关系描述前后矛盾 | 一会儿说是朋友，一会儿说是敌人 |
| 关系空洞 | 关系没有具体表现 | 说是好朋友，但没有任何互动 |
| 关系失衡 | 关系不对等 | 一方付出太多，另一方毫无回应 |

## 三、叙事视角问题

### 3.1 常见视角越界

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 内心越界 | 展示非视角角色的内心 | 主角视角却描写反派的想法 |
| 场景越界 | 描写视角角色看不到的场景 | 主角在A地却描写B地发生的事 |
| 信息越界 | 揭示视角角色不知道的信息 | "他还不知道，危险正在靠近" |
| 感知越界 | 描述视角角色无法感知的事物 | 主角背对着却知道身后的情况 |

### 3.2 常见视角混乱

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 视角切换频繁 | 短时间内多次切换视角 | 一段之内切换3次以上视角 |
| 视角切换无标识 | 切换视角没有明确标识 | 突然从主角视角变成配角视角 |
| 视角混合 | 同一段落混合多个视角 | 一句话中包含两个角色的想法 |
| 视角错位 | 视角角色与叙述者不一致 | 第一人称却描述自己的外貌 |

## 四、伏笔回收问题

### 4.1 常见伏笔问题

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 伏笔遗忘 | 埋下的伏笔没有回收 | 提到的神秘物品再也没有出现 |
| 伏笔突兀回收 | 伏笔回收过于突兀 | 几十章前的伏笔突然回收，没有铺垫 |
| 伏笔无效 | 伏笔回收没有作用 | 回收了伏笔但对剧情没有影响 |
| 伏笔矛盾 | 伏笔回收与前文矛盾 | 伏笔说A，回收时变成B |
| 伏笔过多 | 伏笔太多无法全部回收 | 埋下10个伏笔，只回收3个 |

### 4.2 常见逻辑闭环问题

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 主线未闭环 | 主线故事没有结局 | 主角的目标没有达成 |
| 副线未闭环 | 副线故事没有结局 | 配角的故事线没有交代 |
| 悬案未解决 | 留下的悬念没有解答 | 神秘人物的身份没有揭晓 |
| 结局仓促 | 结局过于仓促 | 一章之内解决所有问题 |
| 结局拖沓 | 结局过于拖沓 | 用过多篇幅描写结局 |

## 五、节奏把控问题

### 5.1 常见节奏问题

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 节奏拖沓 | 情节进展缓慢 | 一章只描写一个简单场景 |
| 节奏仓促 | 情节进展过快 | 几章之内完成大量剧情 |
| 节奏单调 | 节奏缺乏变化 | 从头到尾都是同一种节奏 |
| 节奏断裂 | 节奏突然变化 | 慢节奏突然变成快节奏 |

### 5.2 常见爽点问题

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 爽点间隔过长 | 爽点之间距离太远 | 超过20章没有爽点 |
| 爽点间隔过短 | 爽点过于密集 | 每章都有爽点，失去冲击力 |
| 爽点强度不足 | 爽点不够刺激 | 期待很高，结果平淡 |
| 爽点逻辑薄弱 | 爽点缺乏逻辑 | 主角靠运气获得胜利 |

## 六、文风统一问题

### 6.1 常见文风问题

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 用词不统一 | 同一事物有多种称呼 | 一会儿叫"修为"，一会儿叫"实力" |
| 句式不统一 | 句式风格不一致 | 一会儿简洁，一会儿冗长 |
| 语气不统一 | 叙述语气不一致 | 一会儿严肃，一会儿轻松 |
| 风格断裂 | 风格突然变化 | 前半本写实，后半本玄幻 |

### 6.2 常见AI写作痕迹

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 排比过度 | 连续使用排比句 | "他的眼中闪过愤怒、惊讶、疑惑" |
| 修饰过度 | 过度修饰 | "他深吸一口气，缓缓开口道" |
| 总结过度 | 过度总结 | "这一刻，他明白了很多道理" |
| 情感直白 | 直接描述情感 | "他感到非常悲伤" |
| 比喻陈腐 | 使用陈旧比喻 | "如寒冰般的目光" |

## 七、语病与用词问题

### 7.1 常见语法错误

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 成分残缺 | 句子成分不完整 | "经过努力，使他成功了" |
| 搭配不当 | 词语搭配不合理 | "提高水平"（应为"提升"） |
| 语序不当 | 词语顺序错误 | "讨论并学习"（应为"学习并讨论"） |
| 句式杂糅 | 两种句式混合 | "原因是因为..." |

### 7.2 常用词错误

| 错误用词 | 正确用词 | 说明 |
|----------|----------|------|
| 的/地/得混淆 | 根据语境选择 | "高兴的笑了"→"高兴地笑了" |
| 做/作混淆 | 根据词语选择 | "作作业"→"做作业" |
| 再/在混淆 | 根据语义选择 | "在见"→"再见" |
| 即/既混淆 | 根据语义选择 | "既使"→"即使" |
| 必须/必需混淆 | 根据语义选择 | "必需努力"→"必须努力" |

### 7.3 常见标点错误

| 问题类型 | 描述 | 示例 |
|----------|------|------|
| 引号使用错误 | 引号内外标点错误 | "他说:"→"他说：" |
| 逗号滥用 | 不必要的逗号 | "他，走到了门口" |
| 句号使用错误 | 句号位置错误 | "他说。"→"他说。" |
| 冒号使用错误 | 冒号使用不当 | "原因是:..."→"原因是：..." |

## 八、问题分级标准

### P0：严重问题（必须修改）
- 影响读者理解的逻辑矛盾
- 严重的人物OOC行为
- 无法忍受的视角混乱
- 关键伏笔遗漏

### P1：中等问题（建议修改）
- 逻辑不够严谨但不影响理解
- 人物性格不够稳定
- 节奏不够合理
- 文风不够统一

### P2：轻微问题（可选修改）
- 微小的用词优化
- 句式调整
- 标点规范
- 风格微调

---

## 九、问题优先级排序

1. 情节逻辑问题（影响最大）
2. 人物一致性问题（影响很大）
3. 叙事视角问题（影响较大）
4. 伏笔回收问题（影响中等）
5. 节奏把控问题（影响中等）
6. 文风统一问题（影响较小）
7. 语病与用词问题（影响最小）
